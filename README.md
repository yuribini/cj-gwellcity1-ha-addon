설치 방법

1. 홈 어시스턴트의 Supervisor 패널로 이동합니다.
2. "애드온 스토어" 탭을 클릭합니다.
3. 우측 상단의 세 개의 점을 클릭하고 "저장소"를 선택합니다.
4. 다음 저장소 URL을 추가합니다: https://github.com/yuribini/cj-gwellcity1-ha-addon
5. 애드온 스토어에서 "CJ Gwellcity1 애드온"을 클릭합니다.
6. "설치" 버튼을 클릭합니다.
